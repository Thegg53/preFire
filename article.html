How to Use This Conversion Tool

This repository contains a Python script that can fetch and convert note.com articles to standalone HTML files with all images downloaded locally.

Prerequisites

  • Python 3.6 or higher
  • Required packages: requests, beautifulsoup4

Installation

pip install requests beautifulsoup4

Usage

python3 fetch_and_convert.py

The script will:

  1. Fetch the article from the specified URL
  2. Extract the title, content, and metadata
  3. Download all images to the images/ folder
  4. Create a standalone article.html file with proper styling
