Open Source

This website is fully open source. The source code, decklists, ban list, event listings, and card data are all maintained publicly on GitHub.

Anyone is welcome to contribute — whether that means submitting a new deck, reporting a bug, suggesting a card for the watchlist, or improving the site itself.

How to Contribute

All contributions are made through the GitHub repository. Here's what you can do without any coding knowledge:

  • This isnt working at the moment, due to github workflows and such. For now, edit RESOURCES/data/lists/decks.json Add or update a decklist — Add a .txt file to INPUT/decklists/ following the format in the README. A GitHub Action will automatically process it.
  • Suggest a ban or watchlist change — Edit INPUT/banlists/banlist.js or INPUT/banlists/watchlist.js and open a pull request.
  • Submit an event — Add an entry to INPUT/eventlist/events.txt following the format in the README.
  • Report a bug or suggest a feature — Open an issue on the GitHub repository.
